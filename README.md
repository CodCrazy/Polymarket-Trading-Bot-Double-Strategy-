# Polymarket Double Bot System

An automated trading system for Polymarket's Bitcoin up/down 15-minute prediction markets, consisting of a search bot and a trading bot that work together to discover markets and execute trades.

## 🏗️ Architecture

This system has two main components:

### 🔍 Search Bot
- **Location**: `Search bot/`
- **Purpose**: Discovers and maintains a database of BTC up/down 15-minute markets
- **Technology**: TypeScript, Node.js, Polymarket Gamma API
- **Schedule**: Runs every hour to fetch new markets

### 🤖 Double Bot
- **Location**: `Double bot/`
- **Purpose**: Automated trading bot implementing a "doubling" strategy
- **Technology**: TypeScript, Node.js, Polymarket CLOB API, Ethers.js, Gnosis Safe
- **Schedule**: Runs every 15 minutes at precise Eastern Time intervals

## 📊 Trading Strategy

The bot implements a contrarian momentum strategy:

1. **Market Analysis**: Monitors the YES token price of currently active 15-minute BTC markets
2. **Decision Logic**:
   - If YES price > $0.50 (bearish market sentiment) → Buy **NO tokens** for the next future market
   - If YES price ≤ $0.50 (bullish market sentiment) → Buy **YES tokens** for the next future market
3. **Position Management**: Automatically redeems winnings when markets resolve
4. **Risk Scaling**: Uses an exponential counting mechanism for position sizing (`2 * Math.pow(2, count)` where count increments on consecutive losses and resets on wins)

## 🚀 Quick Start

### Prerequisites
- Node.js v16 or higher
- npm or yarn
- A Polymarket account with USDC
- Gnosis Safe setup for secure trading
- Environment variables configured (see Configuration section)

### Installation

1. **Clone and setup Search Bot:**
```bash
cd "Search bot"
npm install
npm run build
```

2. **Setup Double Bot:**
```bash
cd "../Double bot"
npm install
npm run build
```

## ⚙️ Configuration

Create a `.env` file in the `Double bot` directory with the following variables:

```env
# Polymarket Integration
PROXY_ADDRESS=your_polymarket_proxy_address

# Gnosis Safe Configuration
SAFE_ADDRESS=your_gnosis_safe_address

# Wallet Configuration
PRIVATE_KEY=your_wallet_private_key

# RPC Configuration
RPC_URL=https://mainnet.infura.io/v3/YOUR_INFURA_KEY
```

### Environment Variables Details

- **`PROXY_ADDRESS`**: Your Polymarket proxy contract address (required for trading)
- **`SAFE_ADDRESS`**: Gnosis Safe address for multi-signature secure trading
- **`PRIVATE_KEY`**: Private key for the wallet that can execute Safe transactions
- **`RPC_URL`**: Ethereum mainnet RPC endpoint (Infura, Alchemy, etc.)

## 🎯 Usage

### Running the Search Bot
The search bot should run continuously to maintain the market database:

```bash
cd "Search bot"
npm start
```

**What it does:**
- Runs immediately on startup
- Fetches BTC up/down markets from Polymarket API
- Updates `polymarket_markets.json` with new markets
- Runs every hour automatically
- Maintains the 120 most recent markets

### Running the Trading Bot
The trading bot executes the automated strategy:

```bash
cd "Double bot"
npm start
```

**What it does:**
- Runs every 15 minutes at Eastern Time intervals: :59:50, :14:50, :29:50, :44:50
- Checks current active market sentiment
- Places orders for the next future market based on strategy
- Redeems positions when markets resolve
- Logs all activities to console

## 🐛 Troubleshooting

### Common Issues

1. **Markets not found**: Ensure the Search Bot is running and has populated `polymarket_markets.json`
2. **Trading errors**: Check your environment variables and wallet balance
3. **Time zone issues**: Verify your system clock is set to the correct time zone
4. **API errors**: Check Polymarket API status and your internet connection

### Logs
Both bots provide detailed console logging. Check the output for error messages and trading activity.
